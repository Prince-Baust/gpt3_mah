## Getting Started

<!-- Prerequisites -->
### Prerequisites

This project uses NPM as package manager. So please install npm globally to run the app


<!-- Run Locally -->
### Run Locally

Clone the project

```bash
gh repo clone Prince-Baust/gpt3_mah
```

Go to the project directory

```bash
  cd gpt3_mah
```

Install dependencies

```bash
  npm install
```

Start the server
```bash
  npm run start
```
